Issue 01

신입사원 초기 잔업 적용 완화 요청

  • 입사 당일부터 22시 잔업 적용 → 1주 적응 후 20시까지로 조정 요청
  • 기존 인원 피로 누적 & 신입 부적응 상충 → 조립팀 시프트 근무 설정 검토 필요
Issue 02

사명(신성오토텍) 기피 현상 심각 — 조직 문화 개선 필수

  • 구직자가 사명만 들어도 지원 포기하는 사례 다수
  • 채용 설명 시 상황 충분히 안내, 입사 당일 OJT 실시 후 업무 투입 필요
  • 장기적으로 기초 작업 교육 프로그램 구축 필요
  • 타사 대비 신입 퇴사가 잦고 빠른 점도 이슈로 지적
Issue 03

신입사원 온보딩 동행 문화 필요

  • 식당·시설 위치 파악 어려움 → 2~3일 기존 인원 동행 문화 필요
  • 배치 조의 기존 직원과 멘토/멘티 매칭 제도 도입 권고
Issue 04

기존 근무 이력자 재입사 허용 요청

  • 업무 수행에 문제가 없었던 경력자라면 재입사 허용 검토
Issue 05

사출팀 경력자 구인 난항

  • 2~3년 경력자 구인 자체가 어려운 시장 상황
  • 일부 미충족 경력자에 대해 OJT 후 투입하는 내부 교육 프로그램 필요
